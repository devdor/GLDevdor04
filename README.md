OpenGL demo application showing dynamic shadow mapping, camera spline animation and metaballs (marching cubes) with chrome mapping using cubemap shader. There are a few open source utils used in this application:

* [Glfw](https://github.com/glfw)
* [Open Asset Import Library](https://github.com/assimp)
* [Freetype](https://www.freetype.org/)
* [stb_image](https://github.com/nothings/stb)
